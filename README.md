## App

This app is an Instagram Clone, done with React Js and Firebase. 

Go test the App :  [Intagram-clone](https://instagram-clone-f9e1f.web.app/)

- Google Auth with Firebase
- Email Auth with Firebase
- Responsive Design
- Styled-components
- Hooks
- React-router
